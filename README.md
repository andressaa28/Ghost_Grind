## GHOST GRIND
<img width="1024" height="1024" alt="titulo" src="https://github.com/user-attachments/assets/9b51d945-d80b-447b-8224-0662cb940cb5" />


  O jogo é uma releitura do jogo PacMan.
  O código do jogo utilizado foi do Leonardo Nunes Armelin, <https://github.com/Leonardo-Nunes-Armelim/Bytes_Universe/commit/a443a2682d710d4d9b626cc7113ebba58fa2ac74>
  
  O jogo consta com 5 fases, sendo que a última contém um boss, também uma tela inicial, uma tela de escolha de personagens com 4 personagens, telas de transições, tela de game over e vitória e tela de ajuda. O jogo pode ser jogado com teclado ou joystick, quando clicar na tela de ajuda, ela mostra a opção de escolher o teclado ou o joystick (vai ser mostrado os comandos de acordo com o controle escolhido).
  O personagem principal tem 5 vidas, podendo recuperar uma vida se pegar um coração no mapa. No mapa está espalhado também o freeze, turbo e a poção que permite comer um fantasma. Cada uma aparece na tela de acordo com os pontos coletados. Cada fantasma tem uma pontuação diiferente.
